Questions

Common questions

How is this different from the sight reading trainer?

This one gives you a note name and asks you to find the key. The sight reading trainer shows a note on the stave instead, which is the harder and more useful skill.

How long should I use it for?

Five to ten minutes a day. Short and frequent beats long and occasional for this kind of recall.

Does it work with a MIDI keyboard?

Yes, and it is more satisfying that way, because you are using the real instrument.
