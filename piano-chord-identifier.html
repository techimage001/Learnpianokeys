Questions

Common questions

Can it recognise inversions?

Yes. C, E, G and E, G, C and G, C, E are all identified as C major, because they contain the same notes.

How many notes does it need?

Three for a triad, four for a seventh chord. Two notes is an interval rather than a chord, so it will say so.

Does it work with a MIDI keyboard?

Yes, if your browser supports Web MIDI. Plug in and play the chord.
